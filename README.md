# EcoTrack

EcoTrack is a gamified Android application developed to encourage sustainable habits. By turning recycling into an interactive experience, the app helps users track their environmental impact while staying motivated through challenges and rewards.

Key Features:

Recycling Tracker: Easily log different types of waste (plastic, paper, glass) recycled daily.

Gamification System: Earn points, unlock badges, and level up as you improve your recycling habits.

Daily Challenges: Complete specific eco-friendly tasks to earn bonus rewards.

Progress Analytics: Visual charts and statistics to monitor your positive impact on the environment over time.

Technologies Used:

IDE: Android Studio

Language: Java

Database: SQLite

UI/UX: XML for layouts with a focus on Material Design principles.
